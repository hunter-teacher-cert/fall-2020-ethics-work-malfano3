Below is a list of remote learning tools. The presentation of these tools should be accessible for most/all users.

Accessibility Considerations

  • Contrast Ratio of background to text
  • Larger font size
  • Useable on mobile, tablet, and computer
  • All parts of website can be accessed without mouse (tab key)
  • Descriptive alt text included in each image
  • Proper heading structure used for screen readers
